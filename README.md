
---

## 🚀 How to Deploy (Step by Step)

### Option A — Deploy FREE on Vercel (Recommended)

1. Go to https://github.com and create a free account
2. Create a new repository called `renteasy-zimbabwe`
3. Upload both files (keep the `src/` folder structure)
4. Go to https://vercel.com and sign in with GitHub
5. Click "Add New Project" → select your repository
6. Click "Deploy" — done!
7. You'll get a free URL like: `renteasy-zimbabwe.vercel.app`

**To use a custom domain like `renteasy.co.zw`:**
- Buy the domain at https://zispa.org.zw (Zimbabwe registrar)
- Add it in Vercel under Project → Settings → Domains

---

### Option B — Deploy FREE on Netlify

1. Go to https://netlify.com
2. Drag and drop your entire `renteasy/` folder onto the Netlify dashboard
3. Done! You get a free URL instantly.

---

### Option C — Test Locally (on your computer)

1. Open the `renteasy/` folder
2. Double-click `index.html`
3. It opens in your browser — no installation needed!

## 💡 Features Included

- ✅ Home page with how-it-works section
- ✅ Browse & search properties (8 sample listings)
- ✅ Filter by city, type, bedrooms, price
- ✅ Save favourite properties
- ✅ Property detail page with full info
- ✅ Viewing fee payment flow (EcoCash / InnBucks / Card)
- ✅ Contact unlock after payment
- ✅ Landlord dashboard
- ✅ Add new property listing form
- ✅ Viewing requests management
- ✅ Mobile responsive design

---

## 💰 Business Model Built In

- Tenant pays $5–$20 per viewing request
- After payment, landlord contact is instantly unlocked
- Landlords list for FREE
- You earn every time a tenant wants to view a property
